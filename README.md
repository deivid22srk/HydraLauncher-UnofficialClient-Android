# Hydra Launcher Unofficial Client Android
**Hydra Launcher Unofficial Client Android** is a mobile application that brings the Hydra Launcher experience to your Android device. It allows you to browse game catalogues, manage your profile, and download games directly from your phone.

This project is based on [ReTerminal](https://github.com/RohitKushvaha01/ReTerminal) and utilizes [Aria2](https://aria2.github.io/) as its core download engine to provide fast and reliable downloads.

# Features
- [x] **Hydra Catalogue**: Browse Trending, Weekly, and Achievement categories.
- [x] **"Surprise Me"**: Get random game recommendations with polished UI animations.
- [x] **Global Search**: Find games across the Hydra catalogue with real-time suggestions.
- [x] **Detailed Game Info**: View localized Steam metadata, screenshots, and system requirements.
- [x] **Profile Management**: Manage your Hydra account, edit bio, and upload images.
- [x] **Built-in Terminal**: Powerful terminal emulator based on Termux.
- [x] **Aria2 Integration**: High-performance download management.
- [x] **Multiple Sessions**: Support for multiple concurrent terminal sessions.
